# BXK Trader Pro Principles

## 1. One Source of Truth
Every calculation exists in only one place.

## 2. Modular First
Each engine has one responsibility.

## 3. Decision Support
The application tells the trader what action to take, not just what the market is doing.

## 4. Five-Second Rule
Any important screen should be understandable within five seconds.

## 5. Risk First
Protect capital before seeking profit.

## 6. Professional Quality
Every release should be deployable and documented.

## 7. Simplicity Wins
Prefer clear, maintainable solutions over clever complexity.
